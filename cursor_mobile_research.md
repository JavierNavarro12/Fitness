# Cursor IDE: Uso Móvil y Beneficios de Membresía Pro

## Respuesta Directa a tu Pregunta

**Parcialmente correcto**: Como miembro Pro de Cursor ($20/mes), tienes acceso a casi todas las funcionalidades **excepto Background Agents sin pagar adicional**. Sin embargo, **actualmente NO existe una app móvil oficial de Cursor**.

## 📱 Estado Actual del Uso Móvil

### ❌ **App Móvil Nativa**
- **No existe** app móvil oficial para iOS/Android
- **No hay planes confirmados** para desarrollo móvil nativo a corto plazo
- Cursor es exclusivamente una **aplicación de escritorio** (Windows, macOS, Linux)

### 🌐 **Alternativas Móviles Disponibles**
1. **App Web Cursor** (lanzada en junio 2025):
   - Gestionar Background Agents desde navegador móvil
   - Asignar tareas a agentes remotos
   - Monitorear progreso de trabajos
   - **Requiere suscripción Pro o superior**

2. **Acceso vía VSCode.dev**:
   - Algunos usuarios reportan usar vscode.dev en móvil
   - Funcionalidad limitada sin características específicas de Cursor

## 💰 Beneficios de Cursor Pro ($20/mes)

### ✅ **Incluido en Pro (sin costo adicional)**
- **Unlimited agent requests** (chat ilimitado)
- **Unlimited tab completions** (autocompletado ilimitado)
- **Access to Bug Bot**
- **Access to maximum context windows**
- **Acceso a modelos premium**: GPT-4.1, Claude Sonnet 3.5, Gemini Pro
- **500 fast requests/mes** + **unlimited slow requests**

### ⚠️ **Background Agents - Requiere Pago Adicional**
Aunque Cursor Pro incluye "Access to Background Agents", para usarlos realmente necesitas:

1. **Habilitar "Usage-based billing"** en tu cuenta
2. **Pagar por uso** (modelo de pago por consumo)
3. **Los Background Agents usan "Max Mode"** que es costoso
4. **Requiere conexión a GitHub** para manejo de PRs

### 🔍 **¿Por qué Background Agents Requiere Pago Extra?**
- Son **extremadamente intensivos en recursos**
- Ejecutan **múltiples agentes en paralelo** en la nube
- Usan los **modelos más avanzados** (Max Mode)
- **Crean y manejan branches/PRs** automáticamente
